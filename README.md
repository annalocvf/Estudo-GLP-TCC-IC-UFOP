# Iniciação Científica 

Este repositório reúne os dados e códigos utilizados na análise do mercado de revenda de Gás Liquefeito de Petróleo (GLP) no Brasil, desenvolvida no âmbito de um projeto de Iniciação Científica.

O estudo investiga a estrutura e o funcionamento da cadeia de comercialização do GLP, com foco no segmento de revenda e em aspectos como concentração de mercado, dinâmica de preços e possíveis limitações à concorrência. O trabalho também discute a relevância do GLP como bem essencial no consumo doméstico, especialmente para famílias de baixa renda, para as quais o gasto com o botijão representa parcela significativa da renda.

Este repositório tem como objetivo centralizar e documentar os dados e códigos utilizados para gerar os gráficos e análises do projeto, permitindo que os resultados possam ser reproduzidos, verificados e utilizados em pesquisas futuras.
